## 📚 프로그래밍과 파이썬 소개

### 컴퓨터와 프로그램

**컴퓨터의 특징**
- 대단히 유연한 기계로 다양한 작업 수행 가능
- 리포트 작성, 게임, 데이터 분석 등 폭넓은 활용

**프로그램이란?**
- 컴퓨터가 수행할 명령어(instruction)들의 리스트
- 컴퓨터에게 자세한 작업 지시를 전달하는 문서

### 프로그래밍을 배워야 하는 이유

현대 사회에서 프로그래밍은 필수 기술:
- 보고서 작성 AI 로봇
- 인공지능 알고리즘 주식 매매
- 로봇 프로세스 자동화(RPA)
- 이과/문과 구분 없이 모든 분야에서 필요

**컴퓨터 vs 인간**
- 컴퓨터: 반복적이고 지루한 작업을 빠르고 정확하게 수행
- 인간: 창의적이고 복잡한 문제 해결

## 🐍 파이썬 소개

### 파이썬이란?

1991년 귀도 반 로섬(Guido van Rossum)이 개발한 대화형 프로그래밍 언어

### 파이썬의 특징

1. **생산성**: 간결하고 효율적인 코드 작성 가능
2. **인터프리터 언어**: 작성한 명령문의 결과를 즉시 확인 (초보자에게 적합)
3. **풍부한 라이브러리**: 다양한 기능을 쉽게 사용 가능
4. **쉬운 설치**: 라이브러리 설치가 간편

## 💻 파이썬 기본 사용법

### 1. 대화형 모드 (Interactive Mode)

콘솔에서 문장을 한 줄씩 입력하여 즉시 실행

```python
# 계산하기
>>> 3.14 * 10 * 10
314.0

# 문자열 출력
>>> "나의 " + "고양이"
'나의 고양이'

# 문자열 반복
>>> "Hello" * 10
'HelloHelloHelloHelloHelloHelloHelloHelloHelloHello'
```

### 2. 스크립트 모드 (Script Mode)

파일을 만들어 저장 후 한 번에 실행
- 확장자: `.py`
- 복잡한 코드 작성에 적합

### 3. print() 함수

화면에 문자열이나 계산 결과를 출력

```python
print("파이썬에 오신 것을 환영합니다.")
print("결과값은", 10, "입니다.")

# 구구단 출력 예제
print("3 * 1 =", 3*1)
print("3 * 2 =", 3*2)
print("3 * 3 =", 3*3)
```

### 4. 문자열 (String)

큰따옴표(`"..."`) 또는 작은따옴표(`'...'`) 안에 들어있는 텍스트 데이터

```python
# 올바른 사용
print("Hello World!")

# 오류 발생 (따옴표 누락)
print(Hello World!)  # SyntaxError
```

## 🎨 터틀 그래픽

화면에서 거북이를 이용하여 그림을 그리는 기능

### 기본 예제

```python
import turtle
t = turtle.Turtle()
t.shape("turtle")
t.forward(100)
t.left(90)
t.forward(50)
turtle.mainloop()
turtle.bye()
```

### 삼각형 그리기

```python
import turtle
t = turtle.Turtle()
t.shape("turtle")
t.forward(100)
t.left(120)
t.forward(100)
t.left(120)
t.forward(100)
turtle.mainloop()
turtle.bye()
```
